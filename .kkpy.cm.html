kkpy.cm.doppler(levels=None)¶

KNU Doppler velocity colors.

Examples

>>> cmap = kkpy.cm.doppler()
>>> pm = ax.pcolormesh(lon2d, lat2d, VD2d.T, cmap=cmap['cmap'], norm=cmap['norm'])
>>> plt.colorbar(pm, cmap=cmap['cmap'], ticks=cmap['tick'])
Parameters

levels (array_like) – Array containing user-defined color tick levels. It should have lower and upper bounds.

Returns

dict_cmap – ‘cmap’: matplotlib colormap ‘norm’: color tick levels for plot ‘ticks’: color tick levels for colorbar

Return type

dictionary
